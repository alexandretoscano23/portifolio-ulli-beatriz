# 🎥 Portfolio | Ulli Beatriz - Video & Storymaker

Este é o projeto oficial do portfólio da **Ulli Beatriz**, desenvolvido para apresentar seus trabalhos em produção audiovisual e criação de conteúdo estratégico para stories. O foco principal é a entrega de uma interface de alto nível estético, unindo design minimalista de luxo com performance técnica superior.

### Link do portfolio :
https://portfolio-ulli-beatriz.vercel.app/

## 🛠️ Tecnologias Utilizadas

- **Framework:** [Next.js 14+](https://nextjs.org/) (App Router)
- **Linguagem:** [TypeScript](https://www.typescriptlang.org/)
- **Estilização:** [Tailwind CSS](https://tailwindcss.com/)
- **Banco de Dados:** [Supabase](https://supabase.com/)
- **Animações:** [GSAP](https://greensock.com/gsap/)
- **Componentes:** [Embla Carousel](https://www.embla-carousel.com/)

## 📂 Organização do Projeto

A estrutura segue o padrão modular do Next.js para facilitar a manutenção e escalabilidade:

```text
src/
├── app/          # Rotas, Layouts e Painel Administrativo (/adm)
├── components/   # Componentes da interface (Header, Footer, DivLine)
├── lib/          # Configurações de API e Clientes (Supabase)
├── types/        # Definições de Interfaces TypeScript
└── hooks/        # Lógica personalizada para tratamento de mídias
